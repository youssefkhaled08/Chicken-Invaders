# Chicken-Invaders

A Simple Version of **Chicken Invaders** Implemented Using C++ and SFML.


# Features:

- Multi-Player.
- 3 Levels With 3 Levels Of Difficulty.
- 3 Types of Weapons Which Can be Upgraded.
- Leaderboard For Highscores.
